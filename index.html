My overall Capstone Project consists of an “user” class that houses the various member functions that perform the various operations.
The API library consists of 9 main methods.

  1. The open() method collects user input to open a new user account on file.
  2. The display() method displays the user input information for user account.
  3. The edit() method allows the user to edit all of their account information, excluding their account number.
  4. The add() updates the deposit amount added within each user account balance.
  5. The sub() method updates the deposit amount subtracted after user successfully withdraws money.
  6. The review() method prints all user accounts on file in tabular format including account number, name, type, and balance amount.
  7. The accountNum() method returns the account number of a specific bank account.
  8. The balAmt() method returns the current balance amount of a user account.
  9. The accType() returns the type of account for each account.
