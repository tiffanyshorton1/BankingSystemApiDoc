user::review()

void review() const;

Functionality

This function displays all user account data on file in tabular format.
Information includes:

  • Account number
  • Account holder name
  • Type of Account (Checkings or Savings)
  • Initial addosit amount

    • Parameters

    none

    Return value

    none

    Example

    #include < iostream >
using namespace std;
         
int main()
{
  cout << "Enter account number:" << endl;
  cin >> accNum;
  cout << "Enter account holder name:" << endl;
  cin >> username;
  cout << "Enter kind of account (C/S):" << endl;
  cin >> kindOfAcc;
  cout << "Enter deposit amount:" << endl;
  cin >> deposit;
 
  user ba;
  ba.review();
  
  return 0;
}
